## 1. 웹사이트 개요

- **총 58개 개념 카드**를 7개 영역(A~G)으로 분류하여 카드 그리드로 제시.
- 각 카드는 **썸네일 이미지 + 개념 정의 + 핵심 인사이트 + 교실 적용 + 출처**로 구성되며, 카드 클릭 시 모달로 상세 표시.
- 카드 이미지는 원격 1~7과정, 집합 8~13과정 교안 슬라이드에서 발췌.
- **표시 이미지는 각 카드의 `.thumb <img>` 인라인 썸네일**이며, 모달도 이 썸네일을 우선 사용(`window.__IMGS` 맵은 폴백).
  - ⚠️ 참고: `__IMGS` 맵(card_1~55)과 실제 표시 썸네일은 서로 다른 이미지인 경우가 많음. **교정은 썸네일 기준으로 수행함.**
